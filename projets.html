Projet de Groupe - Mise en place d’une infrastructure pédagogique SISR

2024 Virtualisation Administration réseau Gestion de parc

Contexte :

Création d'une infrastructure réseau complète dédiée à une salle pédagogique pour les étudiants SISR.

Réalisation :

  • Mise en place de 10 postes clients filaires et Wi-Fi
  • Installation et configuration de VMware ESXi sur un serveur physique
  • Déploiement d’une VM Windows Server 2019 pour héberger GLPI et un serveur CS2 (Counter-Strike 2)
  • Configuration réseau avec routeur Cisco et plan d’adressage DHCP
  • Déploiement de GLPI avec l’agent sur chaque poste pour la supervision du parc
  • Installation et configuration du serveur de jeu CS2 et scripts personnalisés (start.bat, server.cfg)

Outils utilisés :

VMware ESXi Windows Server 2019 GLPI Agent GLPI XAMPP Cisco CLI Steam / CS2 Packet Tracer Putty
